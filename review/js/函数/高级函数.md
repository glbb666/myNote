### 函数柯里化

柯里化是一种将使用多个参数的一个函数转换成一系列使用一个参数的函数的技术。

```javascript
function add(a,b,c) {
        return a + b + c;
    }
function curry(fn, ...args) {
    return function(){
        var _args = [...args,...arguments];
        return _args.length<fn.length?curry.call(this, fn, ..._args):fn.apply(this,_args);
    }
}
var add = curry(add,1)
```

### 纯函数

纯函数是指相同的输入总会得到相同的输出，并且不会产生副作用的函数。
