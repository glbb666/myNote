`padding`和`margin`都是相对于父元素的宽度来计算的，我们可以利用这一属性来实现我们的需求。

```html
<div class="scale"></div>
<style>
    .scale {
        width: 100%;
        height: 0;
        padding-bottom: 50%;     
    }
</style>
```

这其中的关键点就是`height: 0;`和`padding-bottom: 50%;`。

我们将元素的高度由`padding`撑开，由于`padding`是根据父元素宽度计算的，所以高度也就变成了相对父元素宽度，同时要将`height`设置为 0，这是为了将元素高度完全交给`padding`负责。
